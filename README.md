# 强柯 · kekeyumuku

> 前进之前先回顾

构建工具和框架，用于**改进判断力** — 不是处理更多信息，而是从信息中提取更好的信号。

---

## 当前关注

- **判断层工具** — 评估内容、论点和决策的质量，而非只做摘要
- **研究系统** — 将工作流转化为可复用的持久资产
- **基础设施思维** — 先问"这个框架能支撑什么"，再问"现在该做什么"

---

## 代表项目

**[worth-learning](https://github.com/kekeyumuku/worth-learning)**
内容判断技能。回答"这值得学吗"，而不是"这说了什么"。

**[crcl-tracking-template](https://github.com/kekeyumuku/crcl-tracking-template)**
CRCL 定性追踪框架。区分利差模型与平台模型的转型信号。

---

*只有改变基础经济逻辑、上行验证或竞争格局的信息，才值得正式记录。*
