Building tools and frameworks for **better judgment** — not processing more information, but extracting better signals from it.

---

## Focus

- **Judgment layers** — evaluate content, arguments, and decisions for quality, not just summary
- **Research systems** — turn workflows into durable, reusable assets
- **Infrastructure-first thinking** — ask "what does this framework support?" before "what should I do now?"
